Masters Thesis

In the process of completing my Interdisciplinary Computer Science Masters Thesis at Mills College!

I am currently researching for my Thesis project. One possible topic I may focus on is related to Software Testing for Beginners. Software Testing is one of the most important aspects in the Software Development Life Cycle and is a skill all software developers should have. Individuals new to programming and the development process have little to no knowledge of how to test code and there is scant resource on software testing tutorials on a beginner's level. I hope to create a repository of software testing tutorials that will provide the significance of testing, testing tutorials that are easy to follow.

The Nonsense Calculator

A Pure JavaScript Calculator Application with nonsense formulas.
